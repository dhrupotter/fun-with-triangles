# Fun with Triangles

## About 🤔

This is a fun application based on some fundamental principles of triangles. The user can:

- Check what combinations of angles create a triangle
- Calculate Hypotenuse
- Calculate Area
- Play fun Quiz

## Tools and Tech 🛠

- **Languages used:** HTML, CSS and JavaScript
